## О проекте

**Language Trainer** — это лёгкое консольное приложение на Lua, которое помогает учить английские слова с русским переводом.  

Создавай свои словари (блоки), добавляй слова, повторяй их в случайном порядке — всё максимально просто, без рекламы и лишних функций.

### Основные возможности

- Создание неограниченного количества блоков слов
- Добавление новых слов в любой блок
- Просмотр содержимого блока
- Режим повторения с проверкой перевода
- 3 готовых стартовых блока с самыми нужными словами
- Полностью на чистом Lua — без внешних библиотек
